Coffee Time Smart System
فكرة المشروع:
موقع ويب تفاعلي وبسيط مخصص لمقهى ذكي (Coffee Time)، يتيح للمستخدمين تصفح قائمة المشروبات المتاحة (مثل اللاتيه والكابوتشينو والإسبريسو) وإضافتها إلى سلة المشتريات مباشرة مع تحديث تلقائي لعدد العناصر في السلة وإظهار تنبيهات تفاعلية فورية.

 الأدوات المستخدمة:
* **HTML5:** لبناء الهيكل البرمجي الأساسي لصفحة الموقع وعناصر القائمة والأزرار.
* **CSS3:** لتنسيق الألوان، الخطوط، وتوزيع العناصر بشكل متناسق وجذاب يناسب هوية المقاهي.
* **JavaScript:** لإضافة الميزات التفاعلية وحساب عدد المشروبات في السلة وتفعيل رسائل التنبيه للمستخدم.
